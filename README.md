1.- Titulo y descripción general
-Nombre del sitio web: Korü
-Contexto y propósito: Una base de datos colaborativa con recetas auténticas de todas las culturas — ingredientes, técnicas, videos y nutrición de más de 190 países.

Resuelve el problema de que las recetas podrían estar esparcidas en diversos sitios y tal vez podría ser algo confuso o enredadizo de encontrar de primeras. Va dirigido a todo tipo de público con tal de que presenten una previa disposición o interés por la cocina o preparación de alguna comida. Da una solución sencilla a aquellos que quieran expandir su conocimiento culinario.


2.- Tecnologías utilizadas
-Teconología frontend: React
-Componente de estilos: Tailwind CSS
-Otras herramientas de desarrollo: Visual Studio Code, Vite, Github, Extensión "Live Share"


3.- Documentación de la API consumida
-Nombre de la API elegida: TheMealDB
-Endpoints principales consumidos: /search.php?s={name} (nombre de la receta), 
/filter.php?c={category} (Categoría culinaria)


4.- Guía de instalación y ejecución
-Requisitos previos: Versión 24.18.0 de node.js
-Instalación: npm install
-Ejecución: npm run dev

Páginas del sitio

Home — Página de inicio con hero, cocinas del mundo y platillos destacados
Recetas — Listado completo de recetas obtenidas de la API
Filtrado — Búsqueda y filtrado por nombre, categoría y país
Detalle — Vista completa de una receta con ingredientes, instrucciones y video
API — Información sobre TheMealDB y sus endpoints
Equipo — Presentación del equipo de desarrollo
Aleatoria — Página anexa que genera una receta sorpresa
